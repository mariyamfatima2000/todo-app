# Todo App

A simple and clean Todo application built using vanilla JavaScript.  
Users can add, complete, and manage daily tasks with data persistence using localStorage.

## Live Demo
https://mariyamfatima2000.github.io/todo-app/

## Features
- Add and delete tasks
- Mark tasks as completed
- Clear completed tasks
- Data persistence with localStorage

## Tech Stack
- HTML
- CSS
- JavaScript

## Project Type
Self-initiated frontend project
